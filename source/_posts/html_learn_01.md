## 缩写规则

- 删去元音字母AEIOU

  image 变成 img

- 只使用首字母

  anchor变成a；ordered list 变成 ol；unordered list 变成 ul

- 只使用开头几个字母

  preview 变成 pre；navigation 变成 nav

## 最典型的标签

- div

  无语义的块级（block）标签

- span

  无语义的内联（inline）标签

- img

  inline-block 的可替换标签

## 快速理解其他标签

- p = div + 段落语义 + 段落样式
- article = div + 文章语义
- section = div + 章节语义
- ol = div + 有序列表语义 + 列表约束



- strong = span + 内容重要语义
- em = span + 语气加强语义
- a = span + 链接语义 + 链接样式 + 跳转行为
